# Order of KALA - Website

Official website for the Order of KALA - an open, borderless, decentralised creative experiment.

## Structure

### Main Pages
- **index.html** - Homepage with manifesto and releases
- **timelines.html** - Interactive generative clock art piece
- **squared.html** - 3D generative sculpture with STL export

### Core Styles
- **styles.css** - Main styling and theme system
- **timelines.css** - Timelines page specific styles
- **squared.css** - Squared page specific styles

### JavaScript
- **animations.js** - Page animations and interactions
- **theme-switcher.js** - Light/dark theme toggle
- **clock.js** - Clock visualization for homepage
- **timelines.js** - Timelines page controls
- **squared.js** - Squared page controls
- **boid-adapted.js** - Boid class for clock animation
- **row-adapted.js** - Row class for clock animation

### Scripts
- **background/** - Three.js animated background
  - background.js
  - style.css
- **Timelines-main/** - Timeline clock sketch files
  - sketch.js
  - row.js
  - boids.js
- **Squared/** - 3D sculpture sketch
  - sketch.js

### Assets
- **logo.svg** - KALA logo
- **discord-icon-svgrepo-com.svg** - Discord icon
- **instagram-svgrepo-com.svg** - Instagram icon

## Features

- **Animated 3D Background** - Three.js boids simulation
- **Generative Art** - Two interactive art pieces (Timelines & Squared)
- **Theme System** - Light/dark mode toggle
- **Responsive Design** - Mobile and desktop friendly
- **Interactive Controls** - Time manipulation, 3D rotation, export options
- **STL Export** - Download 3D models for printing

## Technologies

- **Three.js** - 3D graphics and animations
- **p5.js** - Timelines generative art
- **Vanilla JavaScript** - No framework dependencies
- **CSS3** - Modern styling with custom properties

## To Run

Simply open `index.html` in a modern web browser. No build process required.
